自定义

添加你的专属触发词

每个人都有自己习惯的纠正方式，添加你的口头禅让捕捉更精准

📁 文件位置
hooks/handler.js
找到 CORRECTION_KEYWORDS 数组
✨ 示例
// 你的口头禅
"好像不对",
"等等再想想",
"不是我想的",
📝 添加方法
  1. 打开 hooks/handler.js
  2. 找到 CORRECTION_KEYWORDS 数组
  3. 添加你的口头禅，用逗号分隔
  4. 保存文件即可生效
💡 让 Agent 帮你添加
"帮我把 xxx 添加到 handler.js 的触发词"

参考资料

术语解释

🤖 Agent（智能助手）
专门做某类任务的 AI，每个助手独立学习，互不干扰。
📝 Learnings（学习笔记）
AI 的"错题本"。记录纠正、错误、功能需求。
🪝 Hook（监听器）
自动捕捉纠正的"雷达"。说完"不对"就自动记。
⏰ Cron（定时检查）
每小时自动运行的"巡逻员"。发现重复问题就提醒。
🔍 Pattern（模式）
重复出现的问题。同一个错误出现 2 次以上就是"模式"。
📱 Bot（通知机器人）
通过 Telegram/Discord 发送提醒的机器人。
