## 🔑 关键发现

### 1. GitHub Student Pack 其实不需要 .edu 邮箱（重要！）

> 来源：GitHub 官方讨论区（2025-07-27），https://github.com/orgs/community/discussions/167710

**GitHub 现在允许没有学校邮箱的用户上传证明文件**。具体方式：

1. 在申请页面选择 **"I don't have a school-issued email"**
2. 上传以下任一文件：
   - **学生证照片**（含姓名、在读学期）
   - **学费收据 / 选课确认单**（Enrollment Verification）
   - **成绩单**（Transcript）
   - **在学证明信**（Letter of Enrollment）

> ⚠️ **如果你能以任何方式证明自己是学生（哪怕是国内的学信网报告），直接走 GitHub 验证路径，比搞 .edu 邮箱省事 100 倍。**

---

### 2. JetBrains Student Pack 也可接受文件验证

> 来源：JetBrains 官方支持论坛（2025-01-13），https://intellij-support.jetbrains.com/hc/en-us/community/posts/23896582046738

JetBrains 支持学生通过**发送证明文件 PDF** 验证身份，不强制要求 .edu 邮箱。所需文件：
- 含全名和当前学年信息的**学生证明文件**
- 支持格式：PDF

## 方案一：Liberty University 在线申请（唯一稳定 .edu 域名方案）

### ✅ 优势
- 使用真实 `.liberty.edu` 域名，是被各类学生优惠广泛认可的
- 在线项目成熟（250+ 项目），申请流程标准化
- 偶尔有 **$50 申请费全免**的促销活动（2025-03-06、2026-04-02 均有记录）

### ⚠️ 风险与局限
- 学费较高：本科生 **$410/学分**（2025-2026），研究生更贵
- 必须完成申请流程，包括可能的资格审核
- 不能"零成本"获取——即使申请费免了，仍需真正注册选课才能拿到邮箱
- Liberty University 是基督教大学，部分人可能有顾虑

### 💰 费用明细

| 项目 | 金额 |
|------|------|
| 申请费（常规） | $50（有时可免） |
| 本科在线学费 | $410/学分 |
| 全学期（12学分）Block Rate | $2,850 |
| 军事人员优惠 | $208/学分（需资格） |

### 📋 详细操作步骤

**步骤 1：访问申请页面**
- 官网：https://www.liberty.edu/online/apply-online/
- 或直接访问在线项目介绍页：https://www.liberty.edu/online/becoming-a-student/

**步骤 2：选择项目类型**
- **非学位寻求（Non-Degree Seeking）**：适合只想拿邮箱不拿学位的用户
- 搜索 "non-degree" 或 "certificate" 项目
- 官网目录：https://catalog.liberty.edu/undergraduate/admission/expenses-financial-policy-online-program/

**步骤 3：创建申请账户**
- 访问 Liberty 申请门户
- 使用真实个人信息注册账户
- 记录好登录凭证

**步骤 4：填写申请表**
- 个人信息、联系方式、教育背景
- 选择 "Non-degree seeking" 或 "Certificate" 选项
- 如实填写，不需编造信息

**步骤 5：等待录取通知**
- Liberty 录取相对宽松
- 录取后会收到 .liberty.edu 邮箱及 Canvas（LMS）访问权限

**步骤 6：激活学生邮箱**
- 登录 Liberty 学生门户（ASIST）
- 按指引激活 Microsoft 365 学生邮箱
- 格式：`firstname.lastname@liberty.edu`

**步骤 7：使用 .edu 邮箱申请学生优惠**
- GitHub Student Developer Pack：https://education.github.com/pack
- JetBrains Student Pack：https://www.jetbrains.com/academy/student-pack/

> ⚠️ **绕过验证的方法不存在**。Liberty University 是正规大学，必须真实申请、真实录取才能获得邮箱。使用虚假信息可能导致账户被封。

## 方案二：Calbright College（加州免费在线项目）

### ✅ 优势
- **完全免费**（加州居民），无学费
- 全程在线申请，15 分钟完成
- 专注职业培训（IT、商业等）

### ⚠️ 局限
- **.calbright.edu 不是 .edu 域名**
- 大多数学生折扣服务可能不认可
- 需要**加州居民身份**

### 📋 详细操作步骤

**步骤 1：访问官网**
- https://www.calbright.edu/enroll/

**步骤 2：创建 CCCApply 账户**
- 加州社区大学通用申请系统

**步骤 3：填写 Calbright 非学分申请**
- 选择感兴趣的项目（如 IT 方向）
- 完成基础信息填写

**步骤 4：完成 Onboarding**
- 完成学生门户注册
- 获取 @calbright.edu 邮箱

> ⚠️ **重要提醒**：Calbright 的域名是 `.calbright.edu` 而非 `.edu`，且其认证资质与 .edu 学校有区别。大多数国际学生优惠（如 GitHub Copilot）不保证接受此邮箱。

## 方案三：社区大学在线课程注册

### 适用条件
- 美国居民（大多数社区大学对居住州有要求）
- 接受开放招生（Open Enrollment）的学校

### 常见有效方法

**方法 A：Bristol Community College（马萨诸塞州）**
- 免费项目：MassEducate / MassReconnect
- 全州居民可免费就读
- 网址：https://bristolcc.edu/getstartedatbristol/earnyourdegreetuitionfree.html

**方法 B：Los Rios Community College District（加州萨克拉门托地区）**
- 开放招生
- 学费低（$46/学分，部分项目免费）
- 网址：https://losrios.edu/

**方法 C：Berkeley City College（加州）**
- 开放招生，夏季/秋季注册从 4 月中开始
- 网址：https://www.berkeleycitycollege.edu/

> 🔍 **注意**：大多数加州社区大学的邮箱域名是 `.cc.ca.us`，而非 `.edu`。这意味着**大部分学生优惠可能不认可**。在注册前，确认学校是否提供 `.edu` 域名邮箱。

## ⚡ 最佳推荐路径（按用户情况）

### 情况 A：能证明自己在读（包括国内学生）

**直接走 GitHub Student Pack 文档验证路线（$0）：**

1. 准备材料：
   - 有效学生证正反面（英文或有英文标注更好）
   - 成绩单或在学证明
   - 学费收据

2. 申请步骤：
   - 访问 https://education.github.com/pack
   - 登录 GitHub 账号
   - 选择 "I don't have a school-issued email"
   - 上传上述任一证明文件
   - 等待审核（通常 2-5 个工作日）

### 情况 B：非学生、想要 .edu 邮箱

**Liberty University 非学位申请（成本约 $50-少量学费）：**

1. 等待官网 $50 申请费豁免促销活动
2. 申请 Certificate 或 Non-degree 项目
3. 获得 .liberty.edu 邮箱
4. 用该邮箱申请 GitHub、JetBrains 等

### 情况 C：人在美国、想要低成本 .edu

**找开放招生的社区大学注册最便宜的在线课程：**

1. 搜索所在州的 "open enrollment community college"
2. 申请最便宜的 1 学分课程（通常 $30-100）
3. 注册后获得 .edu 邮箱
4. 用完优惠后取消课程（注意退款政策）

## ⚠️ 风险与注意事项

| 风险 | 说明 | 规避方法 |
|------|------|----------|
| 申请费损失 | Liberty $50 申请费不一定能退 | 等促销活动（免费申请期） |
| 学费陷阱 | 注册后即使不修课也可能有费用 | 仔细阅读财务政策 |
| 邮箱失效 | 退学/休学后邮箱可能被收回 | 保持低学费注册状态 |
| 违反服务条款 | 部分平台禁止非学生使用学生优惠 | 了解各平台规则 |
| 虚假信息风险 | 编造学历信息可能违反法律 | 使用真实信息申请 |
