## ⚡ 一句话结论

**B站搜索 API（WBI 签名）无法绕过，但存在两条可落地的替代路径：① Tag API + 热门视频流（完全无需认证）；② Playwright 真实浏览器 + Cookie（模拟登录用户）。**

---

## 🔍 核心发现

### 1. 412 风控根因（更新）

| 防护层 | 2026年状态 | 能否绕过 |
|--------|-----------|---------|
| TLS JA3/JA4 指纹 | curl_cffi 可模拟 | ✅ |
| WBI 签名 | **2025-05-26 全面强制** | ❌ 无法纯本地绕过 |
| buvid3 Cookie | **必填**，匿名请求必 412 | ❌ |
| IP 频率限制 | 国内 IP 更严格 | ⚠️ 可缓解 |
| 浏览器环境检测 | Playwright 可绕过 | ✅ |

### 2. 各方向研究结果汇总

#### ✅ 可用：Tag API（无需 WBI / Cookie）
- URL: `https://api.bilibili.com/x/tag/info?tag_name=关键词`
- 实测: 5/5 次全部 200（稳定）
- 局限：只能查 tag 详情，无法枚举 tag 下的视频

#### ✅ 可用：热门视频流（无需 WBI / Cookie）
- URL: `https://api.bilibili.com/x/web-interface/popular?ps=20&pn=1`
- 完全无需认证，稳定
- 局限：非搜索，是推荐流

#### ❌ 主搜索 API（间歇性 412）
- 即使正确 WBI，仍间歇性 412（实测：5次=200,412,200,200,412）

#### ❌ 搜索引擎 BV 号检索
- Google/Bing/DuckDuckGo/百度/SearXNG 全部返回零结果
- 根因：B站 SPA 架构使 BV 号不在爬虫可见 HTML 中

#### ❌ UP主空间 API
- yt-dlp 实测（2025-01）：需要 WBI + Cookie，同样 412

## 🛠️ 推荐方案

### 方案 B（最简单，立即可做）：热门视频流 + 关键词过滤

```
步骤 1: GET https://api.bilibili.com/x/web-interface/popular?ps=20&pn=1
步骤 2: 遍历视频列表，匹配标题含"关键词"
步骤 3: 提取 bvid 字段
```
- ✅ 完全无需认证，代码极简（2小时内完成）
- ❌ 对冷门长尾关键词覆盖率有限

### 方案 C（最完整，需 Cookie）：Playwright + Cookie

1. 安装 "Get cookies.txt LOCALLY" 插件，导出 B站 Cookie
2. Playwright 加载 Cookie + stealth.min.js
3. 访问 `search.bilibili.com/all?keyword=关键词`
4. 从页面 DOM 提取 BV 号
