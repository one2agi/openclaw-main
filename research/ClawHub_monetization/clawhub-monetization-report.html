ClaWHub 是 OpenClaw 官方 skill registry + marketplace，被比作"AI Agent 的 App Store"。

⚠️ 关键发现

ClaWHub 官方目前没有明确的付费/定价功能界面。GitHub issue #1752（2026年1月）有开发者询问"如何在 ClaWHub 设置付费"，官方未给出功能说明。付费变现更像是一种非正式生态行为——开发者通过外部渠道（GitHub Sponsors、自建 Gumroad/Patreon）实现收入。

三、变现路径

路径1：ClaWHub Premium Skill 销售
定价 $10–$200/个，单个优质垂直 skill 收入 $100–$1,000/月。⚠️ 平台付费功能尚不完善。
路径2：企业定制 Skill 开发
定价 $500–$2,000/项目，适合有特定行业垂直知识的开发者。
路径3：Setup + Consulting 服务（最快见收入）
Setup 费 $200–$500/次，月度托管 $50–$200/月。已有创始人月一实现 $3,600 收入。
路径4：VPS 托管
成本 $4–6/月，收费 $75–$150/月/客户，毛利率 90%+。DigitalOcean、Alibaba Cloud、Tencent、Hostinger 均已入场。
路径5：安全审计工具
企业级需求，$30K+ MRR 潜力。ClaWHub 已有 341 个恶意 skills 被发现，多个 CVE 被披露。
路径6：内容变现
教程、模板包、Affiliate 营销，平台：Gumroad、Patreon、Notion 模板。

四、竞争格局 & 平台审核

竞争激烈程度：低。 生态仍处早期，大量垂直领域空白。头部玩家主要是独立开发者。Snyk、CrowdStrike 等安全厂商已发布 OpenClaw 相关研究报告。

平台审核：ClaWHub 自述 "no gatekeeping"（无门槛发布），有 moderation hooks + vector search。但安全问题值得关注：14,706 个 skills 中有 1,103 个恶意 skills（约7.5%）。

五、局限性

  • 平台抽成比例未确认（官方未公开 marketplace fee 百分比）
  • 付费功能状态存疑——可能付费变现仍非平台原生支持
  • $100–$1,000/月收入数据来源于第三方博客，非平台官方披露
  • ClaWHub 生态发展迅速，数据可能快速变化

🎯 对墨染的建议

  1. 如果要做 ClaWHub skill 变现：优先制作垂直领域 skill（如特定平台集成、特定工作流模板），定价 $10–$50 走薄利多销路线
