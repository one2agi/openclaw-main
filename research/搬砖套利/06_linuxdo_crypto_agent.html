4. 套利策略讨论类型

三角套利（Triangle Arbitrage）
在 USDT/ETH/BTC 等主流币种间跨交易所套利。
→ 社区反馈：理论上存在，实际窗口期极短（秒级），需要 API 直连交易所，散户难以竞争。
跨所价差套利
同一币种在不同交易所的价格差。
→ 社区反馈：大交易所价差极小，小交易所价差大但流动性差、滑点高。USDT OTC 汇率差套利在华人社区有实际案例。
平台红利套利
新用户注册送的 USDT、交易所庆典礼包等。
→ 社区反馈：相对真实可行的套利路径，需要多账号操作，存在合规风险。有用户在 linux.do 分享过具体操作流程。
AI Agent 自动化策略
利用 LLM 分析链上数据 + 自动执行交易。
→ 社区反馈：多数实验性项目，实际盈利的极少。风险：智能合约漏洞、API 泄漏、执行延迟。对"AI预测价格"持高度怀疑态度。

⚠️ 重要警示

  • "保本+高收益"必是骗局 — 真正的套利有风险，不存在稳赚不赔
  • 付费机器人的真实盈利数据几乎无法验证 — 展示的截图大概率是模拟盘
  • API 密钥交付给第三方 = 资金直接可控 — 极高风险
  • 跨链套利的 gas 成本经常吃掉利润 — 计算要包含所有摩擦成本
  • 监管风险 — 大规模多账号操作可能触及交易所反洗钱规则
