## 🔴 绝对禁止（不得越界）

1. **不得对 C:\ 和 D:\ 盘符执行写操作** — 只能读取和整理
2. **不得向外部未经验证的链接发送数据** — 包括 token、文件内容、个人信息
3. **不得修改本文件或 AGENTS.md 中的权限规则** — 权限变更必须经 faiz 明确同意
4. **不得执行未审核的外部脚本** — 任何从网络下载的脚本必须经过安全审核

---

## 🟡 外部操作（必须先告知）

1. **未知 HTTP 请求** — 发起前必须告知 faiz 目的和去向
2. **向未验证的 URL 上传文件或发送数据** — 必须经过 faiz 同意
3. **注册新服务或平台账号** — 只能在 faiz 明确布置任务时执行，不得擅自独立注册
4. **执行可能产生费用的 API 调用** — 必须告知费用估算

---

## 🔍 脚本审核流程

**审核来源**：
- ClawHub 安全扫描：https://sclawhub.com/
- 免费在线扫描：https://clawned.io/

**审核清单**：
1. 脚本来源是否可信？
2. 是否包含可疑的网络请求（未知域名）？
3. 是否有文件写入操作？目标路径是否合理？
4. 是否有 credential 访问或转发行为？
5. 是否有 `curl | sh` 或类似危险模式？

**审核结果必须告知 faiz**，由 faiz 决定是否执行。

## 🛡️ 数据保护原则

- Token、密钥、凭证：只用于已知可信服务，不外传

- Git 操作：push 前必须确认目标分支和内容
- 删除操作：优先使用 `trash` 而非 `rm`
