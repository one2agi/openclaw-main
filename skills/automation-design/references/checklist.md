```

创建日期：_______________________

## 1. 边界定义（必填）
这个任务要做的事：_________________________________
做到什么程度算完成：_________________________________
最大运行时间（估算）：_______________

## 2. 执行者选择（单选）
[ ] 纯脚本（无需 AI）
[ ] Cron + 确定性脚本

## 3. 步骤清单（必填）

最多连续失败多少次：____ 之后怎么办：________________

## 5. 风险评估
最坏结果是什么：_______________________
是否不可逆：_______________________
```

## 失败模式速查

| 特征 | 问题 | 解决方案 |
|------|------|---------|
| "让 AI 做语义分析" | 不确定、不可靠 | 用规则/脚本替代 |
| "跑多久算完成" | 无边界 | 加最大步数限制 |
| "部分失败" | 状态不一致 | 事务性设计 |
| "依赖外部 API" | 不稳定 | 加超时 + 重试 |
| "处理用户数据" | 隐私风险 | 确认 PERMISSIONS.md |
