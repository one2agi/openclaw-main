# 使用指南

## 调用方式

当用户提供小红书图文素材时，使用此 skill 生成生图 prompt。

## 输入要求

**必需：**
- 标题
- 核心内容（1-N 条要点）

**可选：**
- 副标题
- 标签/关键词
- 参考图（我帮你分析风格）

**示例输入：**
```
标题：Self-Improvement Loop
要点：OODA循环、实时捕获、模式检测、A/B/C/D决策
```

## 输出

直接输出英文 prompt，可复制用于生图模型。

## 流程

1. 接收素材
2. 完整性检查 → 补全提示（如需要）
3. 内容分类 → 选择布局模板
4. 生成 → 输出 prompt

## 注意事项

- Prompt 全部英文，利于生图模型理解
- 视觉风格固定（手绘水彩+莫兰迪+3:4）
- 布局根据内容自动适配
- 如需调整风格，额外说明
