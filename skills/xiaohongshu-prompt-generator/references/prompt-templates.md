## 布局类型参考

### 1. 封面型（Page 1）
- 主标题框居中（大圆角卡片）
- 副标题/描述在下方
- 核心要点横排或散落
- 底部版本标注

### 2. SCQA 四宫格（Page 2）
- 2x2 网格布局
- 每卡片有顶部色条区分类型
- 左上：Situation（橙）
- 右上：Complication（棕）
- 左下：Question（绿）
- 右下：Answer（青绿）

### 3. OODA 循环（Page 3）
- 2x2 网格 + 中心圆
- 每格有大字母背景装饰
- 底部横向流程箭头

### 4. 三栏并列（Page 4, 8）
- 三等分横排
- 每栏独立内容
- 可带图标和描述

### 5. 左右对比（Page 5）
- 双栏布局
- 中间 VS 标识
- 各带标签（Before/After）

### 6. 卡片+表格（Page 6）
- 上方三卡片系统介绍
- 下方表格对比功能

### 7. 2x2 决策（Page 7）
- 四格等分
- 每格有大字母圆圈标识
- 结果框在底部

### 8. 流程图型（Page 9）
- 横向节点+箭头
- 下方多卡片示例
- 底部列表组件

### 9. 步骤卡片（Page 10）
- 2x2 网格编号卡片
- 代码块/树形结构
- 底部文件结构

## Prompt 句式模板

### 标题句式
```
顶部标题：[标题名]（大字，醒目位置）
主标题框：大圆角白色卡片，粗深褐边框
```

### 卡片句式
```
[数量]个[形状]白色卡片[布局方式]：
卡片1（[颜色]图标）：
- 大图标：[emoji]
- 标题：[内容]
- 描述：[详细说明]
```

### 标签句式
```
标签：[名称]（[颜色]标签背景）
字母圆圈标：[字母]（[颜色]背景，粗边框）
```

### 底部句式
```
右下角标注：[版本/日期]
底部横向[内容]：[流程描述]
```

### 风格总结句式
```
整体风格：[关键词1]、[关键词2]、[关键词3]
```

---

## 复用建议

1. **标题区**：固定用"顶部标题：大字"句式
2. **卡片区**：根据内容数量选择布局模板
3. **标签区**：颜色对应关系可复用
4. **底部区**：固定"整体风格"结尾句式
