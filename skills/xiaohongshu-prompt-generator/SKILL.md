# 小红书图文生图 Prompt 生成器

## 核心能力

将用户的文案素材转换为可用的生图 prompt，视觉风格固定为治愈水彩手绘风。

```
素材输入 → 文案补全（如需要）→ 布局判断 → 生图 prompt
```

## 视觉风格（固定）

```
背景：#F5EDE0（暖米色）
卡片：#FFFEF9（米白）
边框：4px solid #5D4E37（深褐）
主色：#8B7355（棕褐）
强调：#C4A77D（暖棕）
点缀：#8B9A6D（灰绿）
风格：手绘水彩、圆角卡片、粗边框、柔和阴影
比例：3:4（1080×1440px）
```

## 布局适配规则

| 内容类型 | 推荐布局 |
|----------|----------|
| 1个主题 | 单卡片居中 |
| 2-4个要点 | 2x2网格 或 单列 |
| 3个并列 | 三栏横排 |
| 4个对比 | 左右双栏 |
| 5+个项 | 混合布局（网格+列表）|
| 流程/步骤 | 横向流程 + 编号卡片 |

## Prompt 结构模板

```markdown
小红书干货图片，3:4竖版比例

[背景描述]
[主标题]（大字，醒目位置）
[副标题]（可选）

[内容区布局描述]
[卡片1内容]
[卡片2内容]
...

[底部标注]（可选）
[背景装饰]（手绘圆形元素）

整体风格：[总结描述]
```

## 使用流程

### Step 1: 接收素材

用户提供的素材可能包含：
- 标题
- 副标题
- 正文要点（1-N条）
- 标签/关键词
- 参考图（可选）

### Step 2: 判断完整性

检查必要元素：
- [ ] 标题（必须有）
- [ ] 核心内容（至少1条）
- [ ] 目标受众/用途（可选）

如果素材不完整，提示用户补充：
- "缺少标题，建议用 'XXX'"
- "内容只有2点，建议补充 1-2 条形成 2x2 布局"
- "这是对比类内容，建议加上 Before/After 标签"

### Step 3: 生成 Prompt

根据内容类型选择对应模板，填充内容，输出完整 prompt。

## 示例

**输入素材**：
```
标题：效率工具
要点：快速、精准、自动化
```

**补全后**：
```
标题：效率工具
副标题：快速 · 精准 · 自动化
要点：
1. 快速 - 10秒处理10万字
2. 精准 - AI 智能理解
3. 自动化 - 全程无需人工干预
```

**输出 Prompt**：
```
小红书干货图片，3:4竖版比例

暖米色背景，手绘水彩插画风格

顶部标题：效率工具（大字，醒目）
副标题：快速 · 精准 · 自动化

三栏横向卡片布局：

卡片1（灰绿色图标）：
- 大图标：⚡
- 标题：快速
- 描述：10秒处理10万字

卡片2（暖棕色图标）：
- 大图标：🎯
- 标题：精准
- 描述：AI 智能理解上下文

卡片3（雾蓝色图标）：
- 大图标：🤖
- 标题：自动化
- 描述：全程无需人工干预

每个卡片有大圆角白底、粗深褐边框、柔和阴影
背景有淡淡的手绘圆形装饰（灰绿、暖棕色）
整体风格：治愈系技术流、手绘线条感、专业清晰
```

## 约束

- Prompt 语言：英文（生图模型兼容性）
- 描述风格：具体、可视化、避免抽象词
- 字数控制：每张图 prompt 控制在 300-500 词
- 输出格式：纯文本 prompt，可直接复制使用
