## 一、商品上架SOP（星辰执行）

### Step 1：读取待上架清单

读取「共享文件_待上架清单.md」，筛选状态=待上架的行。

### Step 2：读取商品信息

根据「资料路径」读取对应商品信息.md，获取：
- 商品名称
- 分类标签
- 夸克网盘链接
- 建议定价
- 商品描述（无极已生成初稿）

### Step 3：生成闲鱼商品信息

**标题（暂定公式）：**
```
[品类简称] [具体场景/模型] [数量] [效果词] [行动词]
示例：【MJ提示词】产品摄影专用 200条 拿来即用
```

**描述（暂定模板）：**
```
还在为[具体问题]烦恼？AI提示词帮你一键搞定！

本资料包收录[N]条[模型名][场景]专用提示词，
覆盖[场景1/场景2/场景3]等多种用途，拿来即用，无需调试。

适用人群：电商从业者/设计师/运营人员/自媒体创作者/学生

购买须知：
- 虚拟商品，拍下秒发货
- 发货内容：夸克网盘链接
- 收到后请及时保存，链接永久有效
- 售出概不退换，请确认后再拍
```

**定价（暂定）：**
- 50-100条：¥5-8
- 100-200条：¥8-15
- 200条+：¥15-25
- 多品类综合大礼包：¥25-50

**标签：**
```
#AI工具 #AI提示词 #[具体模型] #用途场景 #虚拟资源 #电子资料
```

### Step 4：执行上架

手动上架闲鱼，或调用自动化工具。

### Step 5：更新文件

1. 更新「共享文件_待上架清单.md」状态 → 已上架
2. 在「共享文件_已上架商品清单.md」新增一行

## 二、违禁词清单（必须遵守）

```
❌ 禁用：
- 极限词：第一/绝对/最好/100%有效/全网最低/最低价
- 违禁词：破解/永久会员/盗版/注册码/序列号
- 其他：微信/QQ/私聊/站外交易/水印/其他平台水印
- 版权：迪士尼/漫威/宝可梦/任何受版权保护角色
```

---

## 三、每日上架上限

- **每天最多5个新商品**，分散时段上架
- 避免集中批量上架触发风控

---

## 四、异常处理

| 情况 | 处理 |
|------|------|
| 资料包文件损坏/缺链接 | 标记「异常」，写在备注栏，等无极处理 |
| 违禁词不确定 | 跳过该商品，继续处理其他，标注「待确认」 |
| 闲鱼限流/封号 | 立即上报（写在已上架清单备注栏），暂停上架 |
| 资料不足/描述有问题 | 标注「待补充」，继续其他，无极会补充后重新通知 |

## 五、自动回复配置（xianyu-agent）

> 由星辰监控运行状态

```
回复原则：
- 聚焦产品使用体验、物流情况、售后服务
- 每句≤10字，总字数≤40字
- 使用「全新」「可小刀」等电商常用词
- 价格谈拢后引导下单：「拍下改价，马上打包」

以下情况不回复：
- 系统自动回复：[去创建合约]、[去支付]、[去评价]等
- 商品无关的问题：「你是谁」「什么模型」「来自哪里」
```
