## 一、项目定位与目标

### 1.1 我是谁

你告诉我背景，我来执行全部流程——找资源、整理、上架、运营，全部由我处理。

### 1.2 最终目标

在闲鱼上建立一个**AI提示词包**虚拟商品店铺，实现半自动化运营：
- 货源：GitHub/Civitai/PromptHero等平台免费优质提示词
- 存储：夸克网盘
- 销售：闲鱼自动上架 + 自动回复买家咨询

### 1.3 量化目标

| 指标 | 目标 |
|------|------|
| 商品上架数量 | 稳定50-100个在架商品 |
| 日上新 | 3-5个（防风控） |
| 单月收益目标 | ¥500-3000（虚拟资料均值） |
| 自动回复 | 100%覆盖咨询消息 |

## 四、单人运营流程（完整版）

我接收你的指令后，独立完成以下全部环节：

```
① 你：给我资源链接或关键词
      ↓
② 我：抓取资源（GitHub clone / wget / curl）
      ↓
③ 我：内容解析 + 分类判断
      ↓
④ 我：文件重命名（按分类规范）
      ↓
⑤ 我：打包 + 上传夸克网盘（kuake-cli）
      ↓
⑥ 我：生成商品信息.md（含标题/描述/定价/标签）
      ↓
⑦ 我：上架闲鱼（手动或调用xianyu-agent工具）
      ↓
⑧ 我：记录已上架商品，更新清单
```

## 五、每一步详解

### Step 1：接收任务

你告诉我：
- 资源链接（GitHub仓库/网页/文件）
- 或关键词（如"midjourney 产品摄影"）
- 或直接说"帮我上一个新款"

### Step 2：抓取资源

**判断类型：**

| 类型 | 方式 |
|------|------|
| GitHub仓库 | `git clone` |
| 单个文件URL | `wget` |
| 网页内容 | `curl` + 解析 |
| 百度网盘/阿里云盘 | 生成下载链接（需你提供） |

**执行命令示例（通过 exec 工具调用）：**
```bash
git clone https://github.com/rockbenben/ChatGPT-Shortcut
```

### Step 3：内容解析 + 分类

读取提示词内容，判断属于哪个分类：
- 📷 图片创作 / 📝 文本内容 / 🎬 视频创作 / 💻 Coding
- 多分类内容拆分存放

### Step 4：文件重命名规范

```
格式：📂 [分类]提示词包-[具体模型/场景]-[数量].md
示例：📷 Midjourney提示词包-产品摄影风格-200条.md
```

### Step 5：上传夸克网盘

**⚠️ 前提条件：**
1. 你提供夸克Cookie（KUAKE_COOKIE）
2. 你确认网盘根目录路径

**上传方式：**
```bash
docker exec kuake-cli kuake upload [本地文件] [网盘目标路径]
```

**每个资料包建议：**
- 压缩为zip，减少散文件
- 保留原始文件，方便后续补充

### Step 6：生成商品信息文档

自动生成 `上架商品信息.md`，包含：

```markdown
## 商品：📷 Midjourney提示词包-产品摄影风格-200条

**标题：** 【MJ提示词】产品摄影专用 200条拿来即用 让你的主图点击率翻倍

**描述模板：**
- 痛点切入：还在为产品摄影烦恼？调色不会，构图不懂，AI来帮你
- 功能说明：本资料包收录200条Midjourney产品摄影专用提示词，覆盖电商主图、静物特写、场景搭建等多种场景
- 使用场景：淘宝主图/亚马逊listing/小红书种草/INS博主
- 购买须知：虚拟商品，拍下自动发货，恕不退换

**价格区间：** ¥5-15
**标签：** #AI工具 #Midjourney #电商摄影 #AI提示词
**资料包路径：** （夸克链接）
```

### Step 7：上架闲鱼

**标题公式：**
```
[品类简称] [具体场景/模型] [数量] [效果词] [行动词]
示例：【MJ提示词】产品摄影专用 200条 拿来即用
```

**雷区避坑：**
- ❌ 禁用极限词：第一/绝对/最好/100%有效
- ❌ 价格禁词：最低价/全网最低
- ❌ 图片水印：不能有其他平台水印
- ❌ 违禁词：破解/永久会员/盗版

### Step 8：记录与维护

更新 `已上架商品清单.md`，记录：
- 商品名称
- 上架日期
- 闲鱼商品链接
- 状态（正常/已售/下架）

## 七、闲鱼运营规范

### 7.1 上架频率
- 每天**不超过5个**新商品上架
- 避免集中时段批量上架

### 7.2 标题规范
- 不超过30字
- 避免极限词、夸大词
- 突出场景和数量

### 7.3 自动回复配置
- xianyu-agent 实时监听消息
- 识别价格咨询/库存咨询/使用咨询
- 按预设Prompt自动回复（已配置在docker-compose中）

### 7.4 风险管理
- 多账号矩阵分散风险（未来扩展）
- 定期更新商品描述，避免千篇一律
- 关注闲鱼官方规则变化
